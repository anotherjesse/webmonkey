updatePeriodically

void updatePeriodically()
    For now, just update every 24 hours. We check a preference to see when we last updated, and then hook a timer to update 24 hours from now. Note that it is still possible, if unlikely, to have concurrent update requests unless the updater is created in a global context.
    Parameters:
      if - false, do not create the timer
